Данное приложение предназначено для преобразования математических выражений из инфиксной формы в постфиксную 
и последующего вычисления значения выражения с заданными пользователем значениями переменных.

Принцип действия:
1) Введите выражение с помощью вспомогательной формы. Используйте доступные переменные, операторы и функции.
2) Преобразуйте выражение в постфиксную форму с помощью кнопок "Выполнить один шаг" или "Выполнить автоматически".
3) Откройте окно вычисления значения с помощью кнопки "Вычислить".
4) Введите во вспомогательной форме значения переменных. При вводе дробных чисел для обозначения дробной части используйте точку.
5) С помощью кнопок "Выполнить один шаг" или "Выполнить автоматически" вычислите значение выражения.
